# Cultivation Game

A premium ($5-10) idle-survivor cultivation game. Vampire Survivors core loop, with Adventure Capitalist (idle), Risk (territorial conquest), and sect management layered on top. Currently in design phase, transitioning to prototype build (R1-R3 scope). Engine: Godot 4.

See `CLAUDE.md` for project conventions and `STATE.md` for current focus (once authored next session).
